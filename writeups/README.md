# Writeups

Walkthroughs of HackTheBox machines, TryHackMe rooms, and CTF challenges I have solved.

> **Important** — HackTheBox writeups for **active** machines are **not** published here
> per [HackTheBox policy](https://help.hackthebox.com/en/articles/5188750-spoiler-policy).
> Notes for active machines stay private; writeups appear after a machine retires.

## Status

Writeups will be added as machines retire. This folder is currently a placeholder.

## Planned format

Each writeup will follow the same structure for consistency:

```markdown
# <Machine name> — <Difficulty> · <OS>

## TL;DR
One-paragraph summary of the attack chain.

## Recon
(nmap output, port-by-port enumeration)

## Initial Foothold
(how user shell was obtained)

## Privilege Escalation
(how root/Administrator was obtained)

## Post-Exploitation
(persistence opportunities, lateral pivots)

## Lessons Learned
(what was new, what mistakes I made)

## References
(blog posts / docs that helped)
```
